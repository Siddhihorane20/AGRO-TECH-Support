# AGRO-TECH Support

This project is a SQL-based system designed to help farmers access agricultural data efficiently. It addresses common real-world challenges faced by farmers, including lack of information about crop details, seasonal data, market prices, and subsidies.

# The system allows:

Storing farmer details and their land information

Managing crop information and seasonal schedules

Tracking market prices of crops over time

Optionally recording crop yields per farmer

# Technologies Used:

SQL/MySQL for database management

Queries for aggregating, filtering, and reporting data

Tables used: Farmer, Crop, MarketPrice, CropYield

# Use Cases:

Farmers can check crop prices over time

Government or agencies can analyze yield and price trends

Students can practice SQL queries and database management
